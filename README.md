#  Music Application

A modern music streaming web application built using **React (Vite)** with **Firebase integration**, featuring authentication, admin dashboard, album browsing, and playlist management.

This project demonstrates scalable frontend architecture, route protection, reusable components, and cloud-based media handling.

---

##  Features

###  User Features
- Browse albums and songs
- Music player UI with controls
- Authentication (Login / Signup)
- Protected routes
- Responsive UI design

---

###  Admin Features
- Admin dashboard
- Upload songs and albums
- Manage music library
- Role-based access control

---

###  Authentication
- Firebase authentication
- Secure login & signup
- Route protection for private pages
